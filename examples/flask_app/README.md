A simple Flask website that demonstrates how to serve images generated by the
`tex2image` client.

You will need to start a `tex2image` server (running on `localhost:8000`) before
running this example.
The easiest way to do this (assuming you have Docker installed) is to run

```bash
docker run --rm -p 8000:8000 -it ghcr.io/olympiad-bot/tex2image
```

After starting the `tex2image` server, open a new terminal, navigate to this
folder and then run

```bash
uv run --extra examples flask run
```

Note that the `--extra examples` flag in this command is to ensure that the
necessary the `flask` library is installed into the virtual environment before
running the example.
